# 🏥 MedPredict — Disease Prediction ML System

Logistic Regression-based diabetes risk predictor with full model evaluation.

## Project Structure

```
disease_prediction/
├── app.py                          ← Flask API (port 8080)
├── requirements.txt
├── model/                          ← Generated by notebook
│   ├── disease_model.pkl
│   ├── scaler.pkl
│   └── model_info.json
├── notebooks/
│   └── disease_prediction.ipynb   ← Jupyter ML pipeline
├── static/
│   └── evaluation_plot.png        ← Generated by notebook
└── templates/
    └── index.html                  ← Light pink frontend
```

## Setup & Run

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the Jupyter Notebook (FIRST!)
```bash
cd notebooks
jupyter notebook disease_prediction.ipynb
```
Run all cells — this trains the model, evaluates it, and saves artifacts.

### 3. Start Flask App
```bash
python app.py
```
Open: http://localhost:8080

## What the Notebook Covers

| Section | Contents |
|---------|----------|
| Data Loading | Pima Indians Diabetes Dataset (768 records, 8 features) |
| Preprocessing | Zero-value imputation, StandardScaler |
| Model | Logistic Regression (class_weight='balanced') |
| Evaluation | Confusion Matrix, ROC-AUC, PR Curve, F1, Recall |
| Feature Importance | Logistic Regression coefficients |
| Cross-Validation | 10-Fold StratifiedKFold |
| Export | Saves model + scaler + metrics to /model/ |

## Why Accuracy Isn't Enough

```
Naive baseline (predict "no diabetes" always):
  Accuracy = 65.1%  ← Looks OK!
  Recall   =  0.0%  ← Misses EVERY diabetic patient ❌

Our Logistic Regression:
  Accuracy = ~75%
  Recall   = ~78%   ← Catches most at-risk patients ✅
  ROC-AUC  = ~83%
```

In medical ML, **Recall (Sensitivity)** and **ROC-AUC** are the primary metrics.
False Negatives (missed disease) are far more costly than False Positives.

## Features

| Feature | Description | Unit |
|---------|-------------|------|
| Pregnancies | Number of pregnancies | count |
| Glucose | Plasma glucose (2hr OGTT) | mg/dL |
| BloodPressure | Diastolic BP | mmHg |
| SkinThickness | Triceps skinfold | mm |
| Insulin | 2hr serum insulin | μU/mL |
| BMI | Body mass index | kg/m² |
| DiabetesPedigreeFunction | Genetic risk score | score |
| Age | Patient age | years |
